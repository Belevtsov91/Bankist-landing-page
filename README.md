# Bankist — Landing Page

Educational landing page project built while learning JavaScript and modern DOM manipulation techniques.

## About

The original design and concept of this project belong to Jonas Schmedtmann (JavaScript course on Udemy). This version has been independently modified, cleaned up, and extended as part of personal learning practice.

## Features

- Smooth scroll navigation
- Sticky navbar via Intersection Observer
- Tabbed operations section
- Scroll-triggered section reveal animations
- Lazy loading images with blur transition
- Testimonials slider with keyboard and dot navigation
- Modal account form
- Fully responsive layout (mobile / tablet / desktop)

## Stack

- HTML5
- CSS3 (custom properties, grid, flexbox)
- Vanilla JavaScript (ES6+)
- Vite (build tool)

## Responsive breakpoints

| Breakpoint | Target |
|---|---|
| < 900px | Tablet |
| < 600px | Mobile |

## Run locally

```bash
npm install
npm run dev
```
